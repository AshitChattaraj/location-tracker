# 🌍 IP Location Tracker

A simple Python GUI application that tracks the geographical location of an IP address and opens the location in Google Maps.

## Features

* Track location using an IP address
* Display City, State, and Country
* Show Latitude and Longitude
* Open location directly in Google Maps
* User-friendly Tkinter GUI
* Fast and lightweight

## Technologies Used

* Python
* Tkinter
* Geocoder
* Webbrowser

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ip-location-tracker.git
cd ip-location-tracker
```

### 2. Install Required Package

```bash
pip install geocoder
```

### 3. Run the Application

```bash
python location_tracker.py
```

## Usage

1. Enter a valid IP address.
2. Click **TRACK LOCATION**.
3. View location details.
4. Click **🌍 Open Google Maps** to view the location on Google Maps.

## Example IP Addresses

* 8.8.8.8
* 1.1.1.1
* 208.67.222.222
